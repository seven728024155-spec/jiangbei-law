## 💡 如何提出产品建议

1. **创建 Issue** - 详细描述你的想法
2. **标签分类** - 使用 `feature`、`enhancement` 等标签
3. **分配给产品经理** - @翟森 会进行评估
4. **等待回复** - 我们会尽快给你反馈
